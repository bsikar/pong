# Pong

A simple GUI Rust Pong game

## Controls

* `w`: Move paddle up
* `s`: Move paddle down

## Play

https://bsikar.github.io/pong/

## Usage

To use this project, simply download or clone the repository, navigate to the root directory of the project and run the following command:

```
cargo run
```
